Создать простое клиент-серверное приложение (а-ля интернет-магазин)
Должно присутствовать:
-	серверная часть,
-	клиентская часть.

Желаемая структура:
Server:
/products
    GET - список продуктов
/product
    GET - параметры продукта
    POST - добавить продукт
    PATCH - изменить продукт
    DELETE - удалить продукт (по id)
База данных: MongoDB

Client:
    Фото
    описание продукта
